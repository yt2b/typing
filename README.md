# typing

NPCと入力速度を競うタイピングゲームです。

<img width="847" height="636" alt="screenshot" src="https://github.com/user-attachments/assets/74c6cce5-0bd8-4230-8f35-e31f37780050" />

## セットアップ

```
git clone https://github.com/yt2b/typing.git
cd typing
npm install
```

## テスト

```
npm run test
```

## ローカルサーバーを起動

```
npm run dev
```

## ビルド

```
npm run build
```
